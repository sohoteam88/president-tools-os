## `/jobs` — Background Workers and Cron Handlers

```
jobs/
├── queues.ts                   # BullMQ queue definitions + Redis connection
│
├── workers/
│   └── transcribe-voice-note.ts    # Whisper transcription worker
│                                   # Trigger: voice note upload complete
│                                   # Output: updates voice_notes.transcript
│
├── crons/
│   ├── daily-journey-reminder.ts   # 20:00 every day — push notification
│   ├── weekly-compile.ts           # Sunday 09:00 — compile drafts
│   ├── monthly-voice-profile.ts    # 1st of month 02:00 — regenerate profiles
│   ├── monthly-anonymize.ts        # 1st of month 03:00 — PDPA auto-anonymize
│   └── webinar-reminders.ts        # Runs frequently — T-24h/T-3h/T-30min/T+5min
│
└── *.test.ts                   # Tests co-located with job files
```

Cron schedules are defined in `vercel.json`:

```json
{
  "crons": [
    { "path": "/api/jobs/daily-journey-reminder", "schedule": "0 20 * * *" },
    { "path": "/api/jobs/weekly-compile", "schedule": "0 9 * * 0" },
    { "path": "/api/jobs/monthly-voice-profile", "schedule": "0 2 1 * *" },
    { "path": "/api/jobs/monthly-anonymize", "schedule": "0 3 1 * *" },
    { "path": "/api/jobs/webinar-reminders", "schedule": "*/5 * * * *" }
  ]
}
```

## Naming Conventions

| Item | Convention | Example |
|------|------------|---------|
| React components | PascalCase | `VoiceRecorder.tsx` |
| API route files | `route.ts` | `app/api/journey/capture/route.ts` |
| Library modules | camelCase | `similarity.ts`, `filter.ts` |
| DB schema files | camelCase | `voice.ts`, `crm.ts` |
| Test files | Same name as subject + `.test.ts` | `filter.test.ts` |
| Environment variables | `SCREAMING_SNAKE_CASE` | `ANTHROPIC_API_KEY` |
| Database tables | `snake_case` | `journey_moments`, `story_bank` |
| Database columns | `snake_case` | `account_id`, `transcript_status` |
| TypeScript types | PascalCase | `JourneyMoment`, `VoiceProfile` |
| API error codes | `snake_case` | `modification_required`, `account_not_found` |
| PostHog events | `module_action` snake_case | `voice_note_recorded` |

## Import Rules

```typescript
// ✅ Use path aliases (configured in tsconfig.json)
import { scopedDb } from '@/lib/db/scoped';
import { VoiceRecorder } from '@/components/voice/VoiceRecorder';
import { runComplianceFilter } from '@/lib/compliance/filter';

// ❌ Never use relative imports that traverse directories
import { scopedDb } from '../../../lib/db/scoped';
```

`tsconfig.json` path configuration:
```json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./*"]
    }
  }
}
```
